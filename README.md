##  Project Overview
This project is a simple web-based **Book Catalog System** that allows users to:
- Register and log in
- View a dashboard
- Add new books
- View the list of books

All data is stored locally using `localStorage`. No backend or database is used.

##  Live Demo
 [\[https://yourusername.github.io/book-catalog/\](https://yourusername.github.io/book-catalog/)](https://github.com/ansyafiq/book-catalog.git)

##  Instructions
1. Register a new account using the **Create Account** page
2. Log in with the same credentials
3. Use the menu to:
   - Add a book
   - View the book list
   - Log out
